This is an instance of a checkdmarc web backend. It is used to check domains (and optionally their mail servers) for email security best practices, including:

  • DNSSEC
  • SPF
  • DKIM
  • DMARC
  • SMTP TLS reporting
  • SMTP-STS
  • BIMI
  • TLS
  • TLSA

checkdmarc is an open source project.
